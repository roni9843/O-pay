# payment-client

Vite + React + Tailwind frontend for the payment example.

Quick start (PowerShell):

```powershell
cd "e:\MY Aplication\3rd Step\88-website\payment-client"
npm install
npm run dev
```

By default the app expects the backend at http://localhost:5000. You can set `VITE_API_URL` in a `.env` file in this folder to change it.
